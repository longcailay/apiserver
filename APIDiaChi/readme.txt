Tạo server API cho combobox chọn khu vực: Tỉnh, huyện, xã

Gõ lệnh: python areasAPI.py  để run server

truy cập:
http://127.0.0.1:5000/areas   => get danh sách tất cả các tỉnh ở việt nam
http://127.0.0.1:5000/areas/vinh-long  => get danh sách tất cả các huyện ở vĩnh long
http://127.0.0.1:5000/areas/vinh-long/huyen-long-ho   => get danh sách tất cả các xã ở huyện long hồ của vĩnh long
